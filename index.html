Gitnesse

Gitnesse is a Cucumber-wiki integration tool. It enables a project to store cucumber features in a git-based wiki, and then test them against your code. Conceptually influenced by Fitnesse.

Installation

Add this line to your applications's Gemfile: 

gem 'gitnesse'
And then execute: 
$ bundle
Or install it yourself as 
$ gem install gitnesse
For Rails 3, create an initializer file config/initializer/gitnesse.rb like this: 
Gitnesse.config do |config|
  config.repository_url = "git@github.com:luishurtado/gitnesse-wiki.wiki"
end

Usage

For Rails 3 apps, there are a few Rake tasks:

  • rake gitnesse:pull will pull features from the remote wiki repository
  • rake gitnesse:push will push new/changed features to the remote wiki repository
